Some projects

Email client - Spring Boot

This project is relies on Spring technology on backend and Android on frontend. Our application is utilizes Java Mail API (for sending/receiving emails and for communicating with email servers) which was integrated in Spring environment, also we used Hibernate ORM to communicate and persist data to MySQL database.

Email client - Android Studio

This project is frontend for our Email Client app. Connection between two projects was realized using Retrofit3 library. The purpose of this project was to give the user access to all of his emails, across all his accounts in one place (Gmail, Yahoo or any other), as well as to enable him to send, sort or even group emails to custom folders.

Warehouse Management - Spring Boot

This is an application that helps us when transferring goods from one warehouse to another. Keeps records of all purchases and shipments of goods from all warehouses of a company.

Student services - Spring Boot

This project is intended for students and professors to keep records of their subjects. It allows the student to register for the exam, allows payments to the account, taking the exam, etc. Professors can grade students, schedule exams, etc.
